## 制作のポイント
### アニメーションをトランジションで作成
Playerのアニメの切り替えには各クリップをトランジションでつないでフラグで管理しました。
トランジションを組むことで、アニメ切り替えが滑らかになり、且つコーディングが効率的なものになりました。
